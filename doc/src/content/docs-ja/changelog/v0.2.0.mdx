`@takazudo/zudo-design-token-lint` v0.2.0 のリリース。

- `classAttributes` 設定オプション: クラス名を抽出する属性名をカスタマイズできるように
- `classFunctions` 設定オプション: クラス名を抽出するユーティリティ関数名をカスタマイズできるように
- `ExtractorOptions` 型をパブリック API に追加
- `extractClasses()` がオプションパラメータ(`ExtractorOptions`)を受け付けるように
- `DEFAULT_CLASS_ATTRIBUTES` と `DEFAULT_CLASS_FUNCTIONS` 定数をパブリック API に追加
- 後方互換性あり — 破壊的変更なし
